DOM Manipulation with form

This DOM Manipulation with form project is a web application that allows users to fill out a responsive registration form and dynamically display the submitted data in a table format. The registration form includes various input fields such as first name, last name, address, pincode, gender (with radio buttons), choice of food (with checkboxes), state, and country (with dropdown select menus).

The JavaScript code handles the dynamic behavior of the registration form. It listens for the form submission event and prevents the default form submission behavior (page reload). The submitted data is displayed in the table with each field corresponding to a specific column. The table is dynamically updated every time a new form submission occurs.
